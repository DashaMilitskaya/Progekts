Реализовать односвязный список студентов (struct Student, внутри есть поля: фамилия, имя, возраст, группа, телефон, массив оценок).
Реализовать действия пользователя (функции): 
+ обычное добавление студента
+ добавление "рандомного" студента.
+ обычное удаление (по номеру в списке, по фамилии),
+ удаление из списка неуспевающих студентов (средняя оценка < 4).
+ вывод всего списка.
+ загрузка списка из файла.txt,
+ из файла.dat (бинарный).
+ выгрузка списка в файл.txt
+ в файл.dat (бинарный).
+ вывод успевающих студентов (средняя оценка >= 4).
+ вывод всех студентов из одной группы.
+ вывод всех студентов с фамилией, начинающейся на "Ив".
